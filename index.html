Andrew's CSS Selectors Cheat Sheet

A quick reference for common CSS selectors

Selector Syntax Example Description
Element p p { color: blue; } Selects all p elements
Class .menu .menu Selects all elements with the class "menu"
ID #header #header {...} Selects the element with the ID "header"
Attribute [type="text"] [type="text"] {...} Selects all input elements with type="text"
