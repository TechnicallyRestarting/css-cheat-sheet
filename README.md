# CSS Selectors Cheat Sheet

A responsive cheat sheet built with HTML and CSS to document common CSS selectors.

## 📌 What This Project Includes
- HTML table structure for organizing data
- CSS styling with custom color palette
- Hover effects for interactive UI feel
- Sticky table header for improved readability
- Monospace formatting for code examples

## 🎨 Design Choices
This project uses a dark theme with warm gold accents to create a clean, readable reference-style interface inspired by developer documentation sites.

## 🧠 What I Learned
- Structuring HTML tables with multiple columns
- Styling tables for readability and usability
- Using CSS hover states and sticky headers
- Organizing a small frontend project for deployment
